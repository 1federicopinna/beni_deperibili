# simulazione_beni_deperibili
In questa repository è contenuto il codice sviluppato per il progetto di simulazione della gestione e distribuzione di una bene deperibile e ottimizzazione delle politiche di riordino.

## file da usare
- SimulatedAnnealing_v2 -> per la parte di SA, l'altro file è quello originale lasciato solo qualora servisse.
- Simulation_con_agenti.py -> per la parte della simulazione considerando gli agenti che generano domanda del bene e le componenti sulla scontistica.
- Simulation_no_agentii.py -> per la simulazione considerando una domanda stocastica con distribuzione normale con media e sigma conoscite, senza agenti.
- AGENT -> codice con le componenti (classi e funzioni) legate alla scontistica.
- Mod_reg_poli -> cartella contente la matrice e i modello di regressione polinomiale.
